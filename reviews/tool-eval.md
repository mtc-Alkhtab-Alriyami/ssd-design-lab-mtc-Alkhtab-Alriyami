# UML Tool Evaluation

This document evaluates the UML tools used in this lab.

---

| Criterion | diagrams.net (draw.io) | PlantUML Web |
|---------|------------------------|--------------|
| Learning curve | Very easy (drag and drop) | Requires learning UML text syntax |
| Version control friendliness | XML-based source, PNG used for reviews | Text-based source, easy to diff |
| Validation support | Manual visual checks only | Built-in syntax validation |
| Collaboration | Easy sharing via uploads and cloud storage | PR and review friendly |
| Export formats | PNG, SVG, PDF | PNG |
| Cost | Free | Free |
